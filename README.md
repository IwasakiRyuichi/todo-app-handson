# TodoApp-Handson

タスク管理アプリケーションのハンズオンプロジェクトです。

## 概要

シンプルなTodo管理アプリケーションです。ブラウザのみで動作し、データはLocalStorageに保存されます。

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML | マークアップ・画面構成 |
| CSS | スタイリング・レイアウト |
| JavaScript | アプリケーションロジック |
| LocalStorage | データの永続化（ブラウザ内保存） |

## 機能

- タスクの追加
- タスクの完了/未完了の切り替え
- タスクの削除
- ページリロード後もデータを保持（LocalStorage）

## 使い方

1. リポジトリをクローン
   ```bash
   git clone https://github.com/IwasakiRyuichi/todo-app-handson.git
   ```
2. `index.html` をブラウザで開く
